HR-Analytics

(R)

This project explores if employee attrition can be predicted by an employees other atributes. To accomplish this I preformed a logistic regression on variables selection from a variable selection algoritm.

Folio

(HTML, CSS, JavaScript, SQL)

Folio is an applicaion that allows teachers to better manage thier students in the classroom. It features a frontend interface that is connected to an SQL backend via a custom API created with Node.js and Express.js.

NBA Player Stats

(R)

This project analyzes the 2018 NBA player stats dataset from Kaggle, and aims to determine if there is a differene in the number of points scored by point gaurds and shooting gaurds. To determine this, I used a two-sample t-test and compared it to a 5% significane level.

Food Tracker

(Java)

Food Tracker is a application that allows students to better manage the food they have in thier homes. This applicaion has a GUI interface built with Swing, and features a full test suite. Persistence is also implemented via JSON, so users can save and load their data.

Mashable Highest Click

(R)

This project explores the News articles dataset from Kaggle, which contains data on thousands of news articles published to the Mashable website. The goal was to see if certain features could be used to increase clicks on an article. To test this, a KNN clustering algorithm was used on training data. The model was then verified with a testing set.
